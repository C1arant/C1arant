# Ciaran Thistle

## About Me

I am a Game Production student based in the United Kingdom, with a focus on immersive systems-driven game design. My work spans from stylized 3D platformers to networked multiplayer environments. I am currently building a persistent space MMO project while continuing to develop tools and workflows for Unity and related technologies.

Key areas of interest include game systems design, multiplayer architecture, content pipelines, and integration of live-service tools such as Twitch and Dockerized server environments.

---

## Skills & Technologies

- **Engines:** Unity (C#), Unreal Engine (Blueprints)
- **Networking:** Unity Relay, Netcode for GameObjects
- **Tooling:** Docker, GitHub Actions, Headless Linux servers
- **Art & Design:** Blender, Substance Painter, Photoshop
- **Streaming & Media:** OBS, Twitch API integration

---

## Current Focus

- **Long-Term Project:** Designing the foundation for a persistent, player-driven space MMO.
- **Technical Development:** Building multiplayer systems, live server deployment workflows, and Unity editor tools for rhythm and FPS games.
- **Ongoing Learning:** Expanding expertise in netcode, procedural content generation, and content pipeline management.
